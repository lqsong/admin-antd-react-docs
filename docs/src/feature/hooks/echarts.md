# useEcharts

 `useEcharts Hooks` 基于 `Echarts` 封装, [官网](https://echarts.apache.org)。

## Props

| 名称     | 类型    | 默认值                      | 说明                                                                           |
| -------- | ------ | -------------------------- | ------------------------------------------------------------------------------------- |
| ref    | React.RefObject  |                      | 显示图表的DOM                                    |
| initOption  | EChartsOption |         | echarts option                                                                    |
| cb | (ec: echarts.ECharts) => any \| undefined  |  |  回调 |
| theme | string  |  |  主题 |
